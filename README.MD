# iGene API Stackstorm pack

## Config

Example:
```yaml
base_url: http://localhost:6000
```

## Actions

ref | description
--- | ---
get_panels       | Get gene panels from iGene
get_worksheets   | Get a list of worksheets or information about one specific
get_sample       | Get information about a sequencing sample


## Requirements
Requires iGene API >=0.5.0
